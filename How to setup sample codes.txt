-----------------------------------------------------------

ASP.NET Programming Bible 2nd Edition 예제 소스 설치방법

-----------------------------------------------------------


- 설치 및 실행방법

1. 압축파일을 풀어 c:\ASPNETBible에 위치시킨다. 
  (다른 디렉토리에 설치해도 상관은 없으나 이 경로에 압축을 풀 것을 권장한다.
   그렇지 않을 경우 프로젝트별 연관관계에 문제가 발생할 수 있다.)

2. intall.hta 파일을 더블클릭하면 설치프로그램이 실행된다.

3. Step 1에서 예제에서 사용하는 데이터베이스를 생성하기 위해 SQL Server에 로그인할 수 있는 계정과 비밀번호를 입력한다.

4. Step 2에서는 ASP.NET Bible 예제 소스에 대한 IIS의 설정을 자동으로 실행한다.

5. 정상적으로 설치된 경우 각 장별로 IIS에 가상 디렉토리가 설정된다.
  (VB.NET의 소스의 경우 'vb__'라는 가상디렉토리명으로, 
   그리고 C# 소스의 경우 'cs__'라는 이름으로 가상디렉토리가 설정된다.)

6. c:\ASPNETBible\source\source\각 장의 폴더명\ 에서 *.vbproj나 *.csproj 파일을
   Visual Studio.NET에서 열어서 소스를 확인할 수 있다.

7. http://localhost/각 장의 폴더명/페이지명.aspx 를 웹 브라우저에서 열어서 소스의 실행결과를 확인할 수 있다.



- 설치시 유의 사항 

1. SQL Server 2000에서 인스턴스를 생성한 경우에는 Step 1 단계에서 데이터베이스 설정이 되지 않습니다. 본 예제 소스는 SQL Server 7.0 사용자를 위해서 인스턴스를 지원하지 않습니다. 
   따라서 SQL Server 2000을 사용하는 시스템의 경우 인스턴스가 없는 Default Server로 설정되어 있어야 합니다.

2. 본 예제의 모든 소스에서 데이터베이스 연결을 위한 연결문자는 편의상 sa 계정에 빈 암호(blank password)를 사용하고 있습니다. 만약 SQL Server의 sa 계정에 패스워드가 설정된 경우 각 소스별로 데이터베이스 연결문자를 수정해야 합니다. 

3. ASP.NET Programming Bible 1판의 예제소스가 설치되어 있는 경우 문제가 발생할 수 있으므로 이전에 설치된 소스는 삭제한 후에 설치하시기 바랍니다.
